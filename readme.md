# Trails
a simple cross-platform mouse trail
### Dependencies

* mingw if you're on windows 10
* glfw3(build from source)

### Running

```
$ git clone https://github.com/ZilDeus/trails YOUR_DIR
$ cd YOUR_DIR
```
* windows
```
$ generate.bat
$ cd winBuild\
$ make run
```
* linux
```
$ generate.zsh
$ cd unixBuild\
$ make run
```
